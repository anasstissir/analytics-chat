# Analytics Chat

An interactive analytics chat interface built with Vue 3 + TypeScript + Tailwind CSS.

## Features

- Chat interface with user/assistant messages
- Renders text, tables, and histogram charts
- Click any user message to edit and regenerate the response
- Light/Dark/System theme toggle
- ChatGPT-style input area

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:5173 in your browser.

## Tech Stack

- Vue 3 (Composition API)
- TypeScript
- Tailwind CSS v4
- Chart.js
- Lucide Icons
